

# INSTALAÇÃO:
sudo apt install  clamav

ou

sudo apt-get install clamav



# APÓS INSTALAR:

-> Resolvendo o problema " ...is locked by another process?"


sudo systemctl stop clamav-freshclam.service

# PRÉ-USO

Atualização dos bancos de dados de vírus:

sudo freshclam


# COMO USAR:

EX; Verifica todos os arquivos do diretorio Raiz;

sudo clamscan -r /


# Verificar um unico arquivo;

clamscan /home/seu-usuario/documento.pdf


# pós verificação -> exibir arquivos detectados/infectados;

clamscan --verbose --infected

sudo clamscan / --recursive | tee clamscan.log


_____________________________________________________

# LIDANDO COM O ERRO "LibClamAV Warning: cli_scanxz: decompress file size exceeds limits " :

sudo clamscan --max-filesize=30M -r -l ClamScanLog -i /

# SALVAR LOG EM DIRETORIO ESPECIFICO

sudo clamscan -r /folder/to/scan/ | grep FOUND >> /path/to/save/report/file.txt

# REMOVER AMEAÇAS:

How do I remove an infected file from Clamscan?
If you would like clamscan to remove the infected file add to the command the --remove option, or you can use --move=/dir to quarantine them. You may also want clamscan to scan larger files. In this case, append the options --max-filesize=4000M and --max-scansize=4000M to the command.6 de mai. de 2023
