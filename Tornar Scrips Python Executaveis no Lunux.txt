Tornar Scrips Python Executaveis no Lunux:



Basta colocar isso na primeira linha do seu script:

#!/usr/bin/env python



Torne o arquivo executável com:

chmod +x myfile.py



# RENOMEIE A EXTENSÃO PARA ".sh"    :


Executar com:

./myfile.sh
