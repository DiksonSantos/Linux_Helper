Melhora a vida da bateria do NoteBook
O TLP -> otimizar a vida útil da bateria, reduzindo o consumo de energia e melhorando o desempenho térmico.
O TLP-RDW -> é um complemento opcional para o TLP. Ele fornece recursos adicionais de gerenciamento de energia para dispositivos de rádio, como Wi-Fi e Bluetooth.

sudo apt install tlp tlp-rdw


Comandos:

INICIALIZADO (?)
sudo systemctl status tlp

CONFERIR - ATIVO ou INATIVO + Informações do Sistema
sudo tlp-stat -s 

CONFIGURAÇÕES
sudo tlp-stat -c

CONFIGURAÇÕES/STATUS
sudo tlp-stat

INFORMAÇÕES
sudo tlp-stat -b

TEMPERATURA da CPU
sudo tlp-stat -t
