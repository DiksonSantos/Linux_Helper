For seeing which gpu is in use:

sudo system76-power graphics

Switch to Nvidia:

sudo system76-power graphics nvidia

sudo reboot

Switch to intel:

sudo system76-power graphics intel

sudo reboot

Switch to  Hybrid Graphics:

sudo system76-power graphics hybrid

sudo reboot
