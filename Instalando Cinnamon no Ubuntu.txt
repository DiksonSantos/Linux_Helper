

Instalando Cinnamon no Ubuntu 18.04 LTS


Para instalar o ambiente Cinnamon no Ubuntu 18.04 LTS, abra um terminal e insira os seguintes comandos:

sudo add-apt-repository ppa:embrosyn/cinnamon
$ sudo apt-get update
$ sudo apt-get install cinnamon
