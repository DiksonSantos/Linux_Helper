

# COMANDO PARA INSTALAR:

Installation pyinstaller: You can pip install the pyinstaller

# COMANDO PARA USAR
pyinstaller --onefile -F --clean -w Programa_em_Python.py


-> Um arquivo ".spec"  será gerado, Use o comando:

pyinstaller Arquivo.spec


-> Um arquivo dentro da pasta "dist" será o executavel agora
-> Use ./Executavel
